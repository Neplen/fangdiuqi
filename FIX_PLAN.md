# BLE 防丢器 APP 问题修复计划

## 待修复的问题清单

### 问题 1:BLE 连接问题（最高优先级）
**现状**: APP 只能连接硬编码的 MAC 地址，无法搜索新设备
**解决方案**:
1. 创建 BleScanner 类用于扫描 BLE 设备 ✅ 已完成
2. 添加设备列表Dialog 供用户选择
3. 支持多设备连接管理
4. 修改数据库支持存储多个设备

### 问题 2:报警铃声问题
**现状**: 铃声无法预览，录音不保存
**解决方案**:
1. 修改 AlarmSoundManager 使用 STREAM_RING 音量流
2. 添加铃声预览播放功能
3. 修复录音文件保存路径

### 问题 3:定时勿扰时间设置
**现状**: 只能设置为当前时间
**解决方案**:
1. 使用 MaterialTimePicker
2. 保存时间到 DataStore

### 问题 4:开启监控闪退
**现状**: 点击开关 APP 崩溃
**解决方案**:
1. 修复前台服务启动逻辑
2. 添加适当的异常处理

### 问题 5:移除地图功能
**解决方案**:
1. 删除 MapFragment 相关文件
2. 更新导航配置
3. 移除地图权限

### 问题 6:权限优化
**解决方案**:
1. 添加忽略电池优化权限请求
2. 添加后台弹出界面权限
3. 引导用户设置自启动
