# 🚀 快速开始 - BLE 防丢器

## 3 分钟上手指南

### 步骤 1：打开项目（30 秒）

1. 启动 **Android Studio**
2. 点击 **Open**
3. 选择本项目根目录（`BleLostFinder` 文件夹）

### 步骤 2：等待同步（1-2 分钟）

Android Studio 会自动：
- 下载 Gradle
- 下载 Android SDK 组件
- 下载项目依赖

**首次打开可能需要几分钟，请耐心等待**

### 步骤 3：编译 APK（1 分钟）

1. 点击顶部菜单：**Build**
2. 选择：**Build Bundle(s) / APK(s)**
3. 点击：**Build APK(s)**

### 步骤 4：找到 APK（10 秒）

编译成功后，APK 文件位于：

```
app/build/outputs/apk/debug/app-debug.apk
```

### 步骤 5：安装到手机

通过 USB 传输 APK 到手机，或使用 adb：

```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

## ✅ 完成！

现在你可以在手机上使用 BLE 防丢器了。

## 功能总览

- 🔵 **蓝牙连接**：自动连接 iTAG 设备
- 📡 **RSSI 监控**：实时监测信号强度
- 🔔 **防丢报警**：超距自动报警
- 📶 **WIFI 勿扰**：连接家中 WiFi 时不报警
- ⏰ **定时勿扰**：设置免打扰时段
- 🔊 **双向响铃**：一键查找设备，按键查找手机
- ⚙️ **完整设置**：所有参数可自定义

## 重要提示

### 首次运行请配置

安装后请进行以下设置：

1. **授予所有权限**
   - 蓝牙权限
   - 位置权限
   - 通知权限

2. **后台保活设置**（重要！）
   - 开启自启动
   - 关闭电池优化
   
   不同品牌手机设置路径不同，详见 README.md

## 使用说明

### 主界面

- 查看设备连接状态
- 查看实时 RSSI 和电量
- 一键查找设备/查找手机
- 开启/关闭监控

### 设置页面

- RSSI 阈值调节（默认 -90dBm）
- 报警延迟调节（默认 60 秒）
- WIFI 勿扰开关
- 定时勿扰设置
- 报警铃声选择
- 录音功能
- 后台保活引导

---

## 注意事项

1. **地图功能**：暂未启用（可选功能）
2. **蓝牙设备**：需要 BLE 4.0+ 设备
3. **系统要求**：Android 8.0+
4. **首次连接**：请确保 iTAG 设备有电
